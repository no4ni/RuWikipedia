## 🎯 Возможности

- Все статьи русской Википедии в виде простых `.txt` файлов.
- Мгновенный поиск и чтение любой статьи по названию.
- Умный просмотр: маленькие статьи показываются целиком, для больших — случайная выборка из 10 строк (70% — строки с ключевым словом, 30% — любые другие).
- Поиск по редиректам и синонимам (например, «Эйнштейн» найдёт статьи и об актёре, и о физике).
- Полная автономность — интернет не нужен.
- Весь архив занимает всего **2.23 ГБ** (в распакованном виде ~13.2 ГБ).

## 🗂️ Содержимое архива `Wikipedia.7z`

После распаковки вы увидите:

- `wikipedia_texts/` — папка со всеми статьями (около 2 млн файлов `.txt`). **Не рекомендуется открывать её в Проводнике** — лучше пользуйтесь программой.
- `wikiget.exe` — главная программа (консольная утилита).
- `canonical_all.txt` — служебный индекс для поиска по синонимам и редиректам.
- `README.txt` — краткая инструкция (дублирует это описание).

## 🚀 Как пользоваться

### Шаг 1. Подготовка
Убедитесь, что на вашем компьютере установлен **.NET 10 Runtime**. Без него программа `wikiget.exe` не запустится.

### Шаг 2. Распаковка
Скачайте архив `Wikipedia.7z` и распакуйте его в любую удобную папку, например:
- `C:\MyWiki`
- `D:\Wikipedia`

Для распаковки подойдёт любой архиватор, поддерживающий формат 7z (например, бесплатный [7-Zip](https://www.7-zip.org/)).

### Шаг 3. Запуск
1. Откройте **командную строку** (`cmd`) или **PowerShell**.
2. Перейдите в папку с распакованными файлами. Например:
   ```cmd
   cd /d D:\Wikipedia
   ```
3. Введите команду:
   ```cmd
   wikiget.exe "Название статьи"
   wikiget.exe "Сознание"
   wikiget.exe "Квантовая механика"
   ```
### Шаг 4. Что вы увидите
Если статья маленькая (до 1.5 КБ), она выведется целиком.
Если статья большая, программа покажет выжимку из 10 случайных строк:
- с вероятностью 70% в выжимку попадут строки, содержащие ваш запрос;
- с вероятностью 30% — любые другие строки из статьи.
Если точное название не найдено, программа сама подберёт подходящие варианты из индекса и покажет случайную статью (например, по запросу «Эйнштейн» может выпасть статья об актёре или физике).

Совет: Если вам нужно прочитать всю статью целиком (даже очень большую), **не используйте Проводник**. Если нужно прочитать статью целиком, воспользуйтесь стандартной командой: 
   ```cmd
   type wikipedia_texts\Название_статьи.txt
   ```

⚙️ Немного технических деталей
Все файлы статей сохранены в кодировке UTF-8 — никаких проблем с русскими буквами.
Программа wikiget.exe написана на C# и скомпилирована в один файл — не требует установки.
Поиск по индексу (canonical_all.txt) происходит быстро: файл читается не целиком, а только до первого совпадения.
